# Netlify

You can use these credentials to authenticate the following nodes:

  • Netlify
  • Netlify Trigger

# Prerequisites

Create a Netlify (opens new window) account.

# Using Access Token

From the Netlify UI:

  1. Navigate to User Settings > Personal Access Tokens.
  2. Select New access token.
  3. Enter a description and select Generate token.

From n8n:

  1. Enter the access token obtained above.
  2. Click Save to create your credentials.
